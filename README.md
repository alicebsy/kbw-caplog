# kbw-caplog develop 브랜치 구분용
쿠폰/장소 스크린샷을 업로드하고 정리/공유할 수 있는 서비스
Spring Boot + Gradle 기반

---

## 🚀 프로젝트 실행 방법

### 1. 환경 준비
- **Java 17+**
- **Gradle Wrapper** (`./gradlew` 사용)
- IDE: IntelliJ IDEA (권장)

### 2. 실행
```bash
# 서버 실행
./gradlew bootRun

# 빌드 (jar 파일 생성)
./gradlew build

# 테스트 실행
./gradlew test
